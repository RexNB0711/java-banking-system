# Java Banking System

## Project Overview

This project is a console-based banking application developed in Java as the final project for my Project-Oriented Programming course. The application simulates a real-world banking system by allowing users to create and manage accounts, perform deposits and withdrawals, transfer funds, and maintain account information through an interactive menu-driven interface.

The project demonstrates object-oriented programming principles including encapsulation, inheritance, polymorphism, and abstraction. It also incorporates file persistence, allowing account data to be saved and loaded between program sessions.

## Features

* Create and manage customer bank accounts
* Deposit and withdraw funds
* Transfer money between accounts
* View account balances and transaction information
* Save account data to files for long-term storage
* Load existing account information when the program starts
* Interactive menu-driven user interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* File I/O and Data Persistence
* IntelliJ IDEA
* Git & GitHub

## Learning Outcomes

Through this project, I strengthened my understanding of software design, object-oriented programming concepts, data management, and version control practices. The project provided hands-on experience developing a larger-scale Java application while applying industry-standard programming principles and problem-solving techniques.
